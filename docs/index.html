About RiyalScript

RiyalScript is a statically-typed, high-speed programming language inspired by C++ and JavaScript, designed specifically for financial applications, trading systems, and business automation. It emphasizes performance, strong typing with type inference, and blockchain/Web3 interoperability.

With RiyalScript, developers can create efficient, low-latency systems that integrate seamlessly with financial platforms and the latest web technologies. The language supports multi-threading, financial data types, and is compiled into optimized bytecode for superior execution speed.

Built with love for the finance and tech community, RiyalScript offers a modern solution for tomorrow's financial computation challenges.

Example Programs

let gain = 1200
let cost = 1000
let roi = (gain - cost) / cost
print(roi)
func add(a, b)
  return a + b
end

print(add(5, 7))
func factorial(n)
  if n <= 1
    return 1
  else
    return n * factorial(n - 1)
  end
end

print(factorial(5))
let price = 450.75
let shares = 20
let total_value = price * shares
print(total_value)
func isEven(n)
  return n % 2 == 0
end

print(isEven(10))
