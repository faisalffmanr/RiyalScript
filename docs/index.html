RiyalScript

High-performance, statically-typed, finance-oriented programming language

What is RiyalScript?

RiyalScript is a modern programming language specifically designed for financial calculations and analysis. It combines the power of static typing with the flexibility of dynamic languages, making it perfect for financial applications, risk assessment, and portfolio management.

let portfolio = [10000, 15000, 20000] let risk = if 0.8 > 0.5 then "High Risk" else "Low Risk" let compound = 1000 * (1 + 0.05) ** 10

Key Features

Financial-First Design

Built specifically for financial calculations with optimized operators and data types.

Quick Start

Installation

# Clone the repository git clone https://github.com/faisalffmanr/riyalscript.git cd riyalscript # Install dependencies npm install # Run examples npm run examples

Basic Usage

import compile from './src/compiler.js'; // Simple calculation const result = compile('let balance = 1000 + 500 * 2', 'js'); console.log(result); // let balance = 2000; // Financial function const func = compile('func calculateTax[income, rate] (income * rate) end', 'js');

Financial Examples

RiyalScript comes with comprehensive example programs that demonstrate every language feature:

1. Portfolio Analysis

let portfolio = [10000, 15000, 20000, 25000] let total = 0 for asset in portfolio do total = total + asset end let riskLevel = if total > 50000 then "High Risk" else "Low Risk"

2. Loan Calculator

func calculateMonthlyPayment[principal, rate, months] ( principal * (rate * (1 + rate) ** months) / ((1 + rate) ** months - 1) ) end let payment = calculateMonthlyPayment[100000, 0.05, 360]

3. Interest Calculator

func compoundInterest[principal, rate, time, frequency] ( principal * (1 + rate / frequency) ** (frequency * time) ) end let compound = compoundInterest[5000, 0.08, 10, 12]

4. Risk Assessment

let stockReturns = [0.05, -0.02, 0.08, 0.03] let riskLevel = if stockReturns[0] > 0.05 then "High Risk" else "Low Risk"

5. Market Analysis

let stockData = {price: 150.25, volume: 1000000, change: 0.02} let recommendation = if stockData.change > 0.02 then "BUY" else "HOLD"

6. Tax Calculator

func calculateTax[income, rate] (income * rate) end let tax = calculateTax[50000, 0.25]

View all examples: GitHub Examples Directory

Compiler Architecture

RiyalScript uses a modern multi-phase compiler pipeline:

  1. Parser: Ohm-based grammar parsing
  2. Analyzer: Semantic analysis with type inference
  3. Optimizer: AST optimization with constant folding
  4. Generator: Clean JavaScript code generation
